2. **10–25 s — Ranking:** switch to default threshold mode and filter to CN + WEB; open “潮汐档案”.
3. **25–40 s — History:** show Bangumi/MAL/composite curves, episode nodes, and the stale MAL snapshot notice.
4. **40–55 s — AI search:** submit “想看一部中日合拍的悬疑 WEB 连载动画”; point out parsed constraints, engine disclosure, reasons, confidence, and latency.
5. **55–75 s — Tier List:** batch-add a title, move it to “夯”, refresh, and show persistence.
6. **75–85 s — Private import:** select the sample CSV, confirm one candidate, and add it locally.
7. **85–90 s — Evidence:** show the README badges, 50-query metrics, test counts, architecture, and phase commit history.

Recording should retain the visible “演示数据模式” banner. Do not imply that fictional records are live source values.
